## Polylang Translation Preview v1.0.0

### What's New
- **Instant Translation Previews**: Select any text in WordPress admin to see live Google Translate tooltips
- **Polylang Integration**: Automatically uses your configured Polylang languages and locales
- **Smart Language Detection**: Distinguishes between en_GB and en_CA, skipping unnecessary translations
- **HTML Preservation**: Maintains formatting (bold, italic, links) in translated content
- **Security First**: Comprehensive Google Cloud API setup with security best practices

### Requirements
- WordPress 5.0+
- PHP 7.4+
- Polylang plugin
- Google Cloud Translation API key

### Installation
1. Download and extract to `/wp-content/plugins/`
2. Activate plugin in WordPress admin
3. Configure Google Cloud API key in Settings ‚Üí Translation Preview
4. Select your default target language from Polylang languages
